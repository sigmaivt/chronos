# Бриф для Claude Design: «Хронос» — Линия жизни на фоне эпох

## Промпт для Claude Design

Скопируй текст ниже и отправь его в Claude Design как первый запрос.

---

Создай лендинг «Хронос» — интерактивный сайт, который помогает человеку осознать своё место в мировой истории.

### Концепция
Пользователь вводит дату рождения. Сайт строит его личную линию жизни на фоне шкалы мировой истории — от древних цивилизаций до сегодняшнего дня. Цель — вызвать ощущение масштаба: как мимолётна и одновременно ценна одна жизнь.

### Навигация — изометрический кубик Рубика
Главный элемент навигации — стилизованный кубик Рубика в изометрической проекции (CSS transforms: skew + rotate, три видимые грани). Каждая грань — сетка 3×3 цветных ячеек. Пользователь кликает ячейки, чтобы «собрать» грань по цвету. Когда грань собрана — она раскрывается с анимацией, открывая раздел сайта.

6 разделов по граням:

**Верхняя грань (вход в опыт):**
- «Начало» — поле ввода даты рождения, кнопка запуска
- «Твоя секунда» — если вся история (5000 лет) = одни сутки, когда ты появился?
- «Эпохи» — вертикальная scroll-лента: Древний мир → Средневековье → Ренессанс → Новое время → Современность

**Левая грань (контекст):**
- «Современники» — карточки великих людей, которые были живы в год рождения пользователя
- «Изобретения» — что уже существовало и чего ещё не было в год рождения

**Правая грань (рефлексия):**
- «Линия жизни» — параллельные шкалы: слева личная жизнь (рождение → сейчас → средняя продолжительность), справа — мировые события, сжатые в ту же шкалу

### Визуальный стиль
- Тёмный фон (#0a0a0f или очень тёмный сине-чёрный)
- Яркие, насыщенные цвета: каждая эпоха имеет палитру:
  - Древний мир — золото, бронза, терракота
  - Средневековье — глубокий синий, серебро
  - Ренессанс — изумрудный, бордо, слоновая кость
  - Новое время — тёмно-зелёный, красное дерево
  - Современность — электрический синий, неоновый пурпур, белый
- Анимации: плавные переходы между эпохами, свечение активных элементов, парящие частицы на фоне (как звёзды)
- Типографика: крупная, контрастная. Заголовки — выразительный шрифт (например, Playfair Display), основной текст — чистый без засечек (например, DM Sans)
- Кубик Рубика — центральный визуальный якорь на hero-секции, с лёгким парящим эффектом (subtle float animation)

### Эстетические референсы
- Histography.io — визуализация масштаба истории через точки-события
- CSS изометрические кубы (CodePen: rotate + skewX + scaleY для трёх граней)
- Тёмные лендинги с неоновыми акцентами (как Stripe, Linear, или космические визуализации)
- Editorial-стиль для секций с текстом (как у The New York Times Company timeline)

### Технические пожелания
- HTML + CSS + vanilla JavaScript (без фреймворков)
- CSS custom properties для палитры эпох
- CSS @keyframes для анимаций (float, glow, fade-in при скролле)
- Изометрический куб через CSS transforms (skewX, rotate, scaleY)
- Responsive — чтобы работало и на мобильном (кубик может упрощаться до плоского меню)

### Что важно
Это учебный проект, поэтому не нужно подгружать реальные данные из API. Можно использовать захардкоженные данные для демонстрации: список великих людей по эпохам, ключевые изобретения, исторические события. Главное — визуальный wow-эффект и интерактивность.
